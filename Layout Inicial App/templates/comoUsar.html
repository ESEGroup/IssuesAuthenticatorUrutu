Cadastramento de dispositivo

Para efetuar o cadastramento de um novo dispositivo, é necessário que o usuário siga os seguintes passos:

Efetuar login utilizando email e senha cadastrados no Issues Monitoring.

Remoção de dispositivo 

        Para efetuar a remoção de um dispositivo cadastrado, é necessário que o usuário siga os seguintes passos:

Efetuar logout no aplicativo.

Alteração de preferências 

        Para efetuar alterações de suas preferências pessoais, é necessário que o usuário siga os seguintes passos:

Efetuar login utilizando email e senha cadastrados no Issues Monitoring. Selecionar opção “Preferências” no menu. Preferências atuais serão exibidas na tela. O ajuste de preferências pode ser feito alterando os valores mostrados. Utilizar botão “Guardar” para confirmar novas preferências.

Consulta de histórico 

        Para efetuar alterações de suas preferências pessoais, é necessário que o usuário siga os seguintes passos:

Efetuar login utilizando email e senha cadastrados no Issues Monitoring. Selecionar opção “Histórico” no menu.
